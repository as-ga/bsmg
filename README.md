# Brain Storming Memory Game (BSMG)

This is a simple memory game implemented using HTML, CSS, and JavaScript. The game involves matching pairs of cards with identical images.

## Getting Started

To get a copy of this project up and running on your local machine, follow these steps:

1. Clone or download this repository.
2. Open the `index.html` file in your web browser.

## How to Play

1. Click on a card to reveal its image.
2. Click on another card to reveal its image.
3. If the images on the two cards match, they remain visible.
4. If the images do not match, both cards will be hidden again after a short delay.
5. Continue revealing cards and matching pairs until all pairs are found.

## Built With

- HTML - For the structure of the webpage.
- CSS - For styling the webpage.
- JavaScript - For implementing the game logic.
